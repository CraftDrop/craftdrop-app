Craftdrop.io API endpoints

By Lucasbolt
GET /api/

Displays the list of available API endpoints and request methods allowed

GET /api/user/

Fetches the user info [uses token authentication]

GET /api/users/

Fetches users info [info is limited]

GET /api/users/<int:pk>/

Fetches user info [info is limited]

GET /api/artist/

Returns all artists available

GET /api/artworks/

Returns all artworks

GET /api/search?q

Returns the search results [artworks can be searched using title, description, or artist's name]

POST /api/register/

Registers user and returns access and refresh tokens in JSON format

POST /api/token/

Generates new JWT tokens [requires login credentials]

POST /api/token/refresh/

Refreshes access token [requires the user's valid refresh token]

POST /api/create_artist/

Creates an artist [requires the user to be logged in - needs access token for validation]

POST /api/logout/

Logs the user out [requires the refresh token]

POST / api/list_artwork

Art listing endpoint [requires authentication and must be an artist]

Pass artwork_id: {'artwork_id': int}

PUT /api/update_profile/

Updates a user profile [requires authentication]

DELETE /api/delete_user/

Delete user account [user must be logged in]

DELETE /api/delist_artwork/

Delist artwork [must be an artist and logged in]

Pass artwork_id: {'artwork_id': int}
